# WasurenBou - わすれん棒

忘れやすい人専用のリマインダーアプリ

## プロジェクト概要

「忘れることは悪いことじゃない。思い出すお手伝いをするだけです。」

### 主要機能
- 音声入力による簡単リマインダー作成
- 段階的通知システム
- 優しいメッセージング

### 技術スタック
- Swift
- SwiftUI
- Core Data
- UserNotifications
- Speech Framework

## 開発環境
- Xcode 16.2+
- iOS 16.0+

## セットアップ手順

1. XcodeでWasurenBou.xcodeprojを開く
2. 必要な権限を設定
   - マイク使用許可
   - 音声認識許可
   - 通知許可
3. シミュレーターまたは実機でビルド
