1. 基本方針

Remind!!!（以下「本アプリ」）は、ユーザーの皆様のプライバシーと個人情報の保護を最重要事項として捉え、以下の方針に基づいてサービスを提供いたします。

プライバシーファースト設計
本アプリは、必要最小限のデータのみを収集し、ユーザーのプライバシーを最優先に考慮して設計されています。

2. 収集する情報の種類

2.1 音声データ

  • 収集目的: 音声入力によるリマインダー作成機能の提供
  • 処理方法: すべてデバイス内で処理（オンデバイス処理）
  • 外部送信: 一切行いません
  • 保存: 音声データ自体は保存されず、変換されたテキストのみがリマインダーとして保存されます

2.2 位置情報

  • 収集目的: GPS連動リマインダー機能の提供（プレミアム機能）
  • 利用条件: ユーザーが明示的に許可した場合のみ
  • 処理方法: デバイス内でのみ処理
  • 外部送信: 一切行いません
  • 保存: ユーザーが設定した特定の場所の座標のみを保存

2.3 アプリ使用データ

  • 収集内容: アプリのクラッシュ情報、基本的な使用統計（匿名化）
  • 収集目的: アプリの安定性向上とバグ修正
  • 個人識別: 個人を特定できる情報は含まれません

2.4 広告関連データ

  • 広告プラットフォーム: Google AdMob
  • 収集データ: 広告配信に必要な最小限の情報
  • ユーザー選択: App Tracking Transparency（ATT）により、ユーザーが追跡の許可/拒否を選択可能
  • GDPR対応: EU圏内のユーザーには追加の同意確認を実施

3. 情報の利用目的

収集した情報は、以下の目的でのみ使用します:

  • リマインダー機能およびチェックリスト機能の提供
  • 音声認識機能の提供（デバイス内処理）
  • GPS連動リマインダー機能の提供（プレミアム機能）
  • アプリの安定性向上およびバグ修正
  • ユーザーサポートの提供
  • 法令に基づく対応（法的義務がある場合のみ）

4. 第三者への情報提供

4.1 広告配信サービス

  • 提供先: Google LLC（AdMob）
  • 提供情報: 広告配信に必要な匿名化された情報
  • 目的: 適切な広告の表示
  • ユーザー制御: 設定により広告の個人化を無効にすることが可能

4.2 プライバシー保護

上記を除き、ユーザーの明示的な同意なく第三者に個人情報を提供することは一切ありません。

5. データの保護と安全性

  • 暗号化: すべてのデータは適切に暗号化されて保存されます
  • アクセス制御: 必要最小限の権限でのみデータにアクセスします
  • セキュリティ監査: 定期的なセキュリティチェックを実施します
  • データ最小化: 必要最小限のデータのみを収集・保存します
  • 自動削除: 不要になったデータは自動的に削除されます

6. ユーザーの権利

ユーザーの皆様には、以下の権利があります:

  • アクセス権: 保存されているデータの確認
  • 訂正権: 不正確なデータの修正要求
  • 削除権: データの削除要求（「忘れられる権利」）
  • 処理停止権: データ処理の停止要求
  • データポータビリティ権: データの他サービスへの移行

これらの権利を行使される場合は、下記お問い合わせ先までご連絡ください。

7. Cookie・類似技術の使用

本アプリは、以下の目的でCookieや類似技術を使用する場合があります:

  • アプリの設定情報の保存
  • 広告配信の最適化（ユーザーの同意がある場合のみ）
  • アプリの利用状況分析（匿名化されたデータのみ）

8. 国際データ転送

広告配信サービス（Google AdMob）の利用に伴い、データが国外に転送される可能性があります。その場合でも、適切な保護措置を講じ、関連法規に準拠します。

9. 未成年者のプライバシー

本アプリは13歳未満の児童を対象としておらず、意図的に13歳未満の児童から個人情報を収集することはありません。13歳未満の児童のデータが誤って収集されたことが判明した場合、速やかに削除いたします。

10. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更やサービス内容の変更に伴い更新される場合があります。重要な変更がある場合は、アプリ内通知やウェブサイトでお知らせいたします。

1. Basic Policy

Remind!!! (the "App") considers the privacy and protection of users' personal information as our top priority and provides services based on the following policy.

Privacy-First Design
This app is designed to collect only the minimum necessary data and prioritizes user privacy above all else.

2. Types of Information Collected

2.1 Voice Data

  • Collection Purpose: Providing voice input reminder creation functionality
  • Processing Method: All processing is done on-device
  • External Transmission: Never transmitted
  • Storage: Voice data itself is not stored; only converted text is saved as reminders

2.2 Location Information

  • Collection Purpose: Providing GPS-linked reminder functionality (Premium feature)
  • Usage Conditions: Only when explicitly permitted by the user
  • Processing Method: Processed only within the device
  • External Transmission: Never transmitted
  • Storage: Only coordinates of specific locations set by users are stored

2.3 App Usage Data

  • Collection Content: App crash information, basic usage statistics (anonymized)
  • Collection Purpose: Improving app stability and bug fixes
  • Personal Identification: Contains no personally identifiable information

2.4 Advertising-Related Data

  • Ad Platform: Google AdMob
  • Collected Data: Minimum information necessary for ad delivery
  • User Choice: Users can choose to allow/deny tracking via App Tracking Transparency (ATT)
  • GDPR Compliance: Additional consent confirmation for EU users

3. Purpose of Information Use

Collected information is used only for the following purposes:

  • Providing reminder and checklist functionality
  • Providing voice recognition functionality (on-device processing)
  • Providing GPS-linked reminder functionality (Premium feature)
  • Improving app stability and bug fixes
  • Providing user support
  • Legal compliance (only when legally required)

4. Information Sharing with Third Parties

4.1 Ad Delivery Service

  • Recipient: Google LLC (AdMob)
  • Information Provided: Anonymized information necessary for ad delivery
  • Purpose: Displaying appropriate advertisements
  • User Control: Ad personalization can be disabled in settings

4.2 Privacy Protection

Except for the above, we never provide personal information to third parties without explicit user consent.

5. Data Protection and Security

  • Encryption: All data is properly encrypted and stored
  • Access Control: Data access is limited to minimum necessary permissions
  • Security Audits: Regular security checks are conducted
  • Data Minimization: Only minimum necessary data is collected and stored
  • Automatic Deletion: Unnecessary data is automatically deleted

6. User Rights

Users have the following rights:

  • Right of Access: Confirmation of stored data
  • Right of Rectification: Request correction of inaccurate data
  • Right of Erasure: Request data deletion ("Right to be Forgotten")
  • Right to Restrict Processing: Request to stop data processing
  • Right of Data Portability: Transfer data to other services

To exercise these rights, please contact us using the information below.

7. Use of Cookies and Similar Technologies

This app may use cookies and similar technologies for the following purposes:

  • Storing app configuration information
  • Optimizing ad delivery (only with user consent)
  • Analyzing app usage (anonymized data only)

8. International Data Transfers

Data may be transferred internationally in connection with the use of ad delivery services (Google AdMob). Even in such cases, we implement appropriate protective measures and comply with relevant laws and regulations.

9. Privacy of Minors

This app is not intended for children under 13 and does not knowingly collect personal information from children under 13. If we become aware that personal information from a child under 13 has been collected inadvertently, we will promptly delete it.

10. Changes to Privacy Policy

This privacy policy may be updated due to changes in laws or service content. In case of significant changes, we will notify users through in-app notifications or our website.
